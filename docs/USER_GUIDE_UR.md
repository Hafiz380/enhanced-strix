# 🦅 Enhanced Strix - آسان انسٹالیشن گائیڈ (Visual Guide)

یہ گائیڈ ان صارفین کے لیے ہے جو ٹیکنیکل مہارت نہیں رکھتے۔ آپ صرف **5 منٹ** میں ٹول انسٹال کر سکتے ہیں۔

## 🛠️ انسٹالیشن سے پہلے کی ضرورت
بس یہ یقینی بنائیں کہ آپ کے کمپیوٹر پر **Python 3.12** انسٹال ہے۔ آپ اسے [python.org](https://www.python.org/downloads/) سے ڈاؤن لوڈ کر سکتے ہیں۔

---

## 🚀 اسٹیپ بائی اسٹیپ طریقہ

### اسٹیپ 1: ون-کلک انسٹالر چلائیں
ٹول کے فولڈر میں جائیں اور `install.bat` نامی فائل پر ڈبل کلک کریں۔

![Double Click Install](https://img.icons8.com/fluency/96/mouse-left-click.png)

### اسٹیپ 2: خودکار سیٹ اپ کا انتظار کریں
ایک کالی ونڈو (Terminal) کھلے گی جس میں پروگریس بار چلے گا۔ آپ کو کچھ بھی ٹائپ کرنے کی ضرورت نہیں، ٹول خود بخود تمام ضروری لائبریریز انسٹال کر لے گا۔

**پروگریس:**
- [██████░░░░] 60% (انسٹال ہو رہا ہے...)

### اسٹیپ 3: کامیابی کا پیغام
جب آپ کو **"INSTALLATION SUCCESSFUL"** لکھا نظر آئے، تو 'Enter' دبائیں۔ اب آپ کے فولڈر میں ایک نئی فائل `Run_Strix_Web.bat` بن جائے گی۔

### اسٹیپ 4: ٹول کا استعمال شروع کریں
بس `Run_Strix_Web.bat` پر ڈبل کلک کریں۔ آپ کا براؤزر خود بخود کھل جائے گا اور آپ ٹول کا خوبصورت انٹرفیس دیکھ سکیں گے۔

---

## ❓ عام مسائل اور ان کا حل

| مسئلہ | حل |
| :--- | :--- |
| **Python Not Found** | پائتھن انسٹال کریں اور انسٹالیشن کے وقت "Add Python to PATH" پر ٹک لگائیں۔ |
| **Network Error** | اپنا انٹرنیٹ چیک کریں کیونکہ ٹول کچھ فائلیں ڈاؤن لوڈ کرتا ہے۔ |
| **Permission Denied** | فائل پر رائٹ کلک کریں اور "Run as Administrator" منتخب کریں۔ |

---

## 🔒 سیکورٹی نوٹ
آپ کی تمام API Keys آپ کے اپنے کمپیوٹر پر محفوظ رہتی ہیں۔ ہم کبھی بھی آپ کا ڈیٹا کسی کلاؤڈ یا سرور پر نہیں بھیجتے۔
